# 2-Lane Step Sequencer

This sequencer features a **dual-lane architecture**, each lane containing **three independent tracks**, allowing for rich and dynamic rhythm creation.

The two lanes can be **mixed together**, enabling layered patterns and complex groove structures.

To keep patterns evolving and avoid repetition, the sequencer includes multiple playback variations:

* Pattern filtering
* Sequential loading
* Random pattern loading
* Alternating lane playback

In addition, sequences can be **automatically shifted over time**, generating unexpected rhythmic variations that go beyond traditional step sequencing.

The goal is to help you discover rhythms you have never heard before.

---

## Features

### Mixer

- **Track Volume Control**: Adjust the volume of each track.
- **Lane Balance Control**: Control the mix between the two lanes.

### Pattern

- **Filtering**: Filter the pattern list.
- **Import / Export**: Save and load patterns.
- **Auto Loading Modes**: Automatically switch patterns in various ways.

### Sequence

- **Shift**: Automatically shift the sequence over time.
- **Copy**: Copy the sequence in 4-step units.

### Sound

- **Attack / Duration**: Adjust the envelope of the sound.
- **Save / Load**: Save and load sound settings.

---

## How to Play

1. **Settings**
   Press the panel button to open the side panel, where all settings are organized in tabs.

2. **Presets**
   **First**, download a preset and import it from the link at the bottom of the Pattern tab.

3. **Play**
   Press the Play button to start the sequence.

4. **Adjust**
   Use the **V** and **P** buttons to adjust velocity and pitch, which are saved to the pattern.

5. **Sound**
   Use the sound name button to adjust sound type, attack, and duration, which can be saved as a sound.

6. For other operations, please refer to the demo video.
